### Dataset Structure
The CSV contains the following fields:
- `identifier` - Unique catalog number
- `designer` - Designer or collection name
- `description` - Brief item description
- `item_type` - Classification (Evening Dress, Day Dress, etc.)
- `collection` - Collection name and year
- `storage_location` - Physical location in archive
- `condition` - Conservation status
- `measurements` - Dimensions
- `image_url` - Link to corresponding image file
